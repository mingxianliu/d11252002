Gemini CLI 斜線命令完整彙整

本文件彙整了 Google Gemini CLI 中所有可用的斜線命令，包括內建命令和自訂選項。

💡 使用說明

  • 在 Gemini CLI 互動模式中輸入 / 會顯示所有可用命令
  • 輸入 /help 可檢視詳細說明
  • Gemini CLI 支援三種命令類型：斜線命令（/）、檔案命令（@）、Shell 命令（!）

⚡ 特殊符號命令

@ 命令（檔案內容注入）

  • @<檔案路徑> - 將檔案內容注入到提示中
  • @<目錄路徑> - 將目錄列表注入到提示中
  • @search <查詢> - 搜尋並引用網路內容
範例：
@src/components/Header.js  # 讀取特定檔案
@docs/                     # 列出文件目錄
@search "React hooks best practices"  # 搜尋網路內容

! 命令（Shell 整合）

  • !<shell_command> - 執行 shell 命令
  • ! (單獨使用) - 切換到持續 shell 模式，再次輸入 ! 退出
範例：
!ls -la           # 列出檔案
!git status       # 檢查 Git 狀態
!npm test         # 執行測試
!                 # 進入/退出 shell 模式

Claude Code 完整快捷鍵與命令指南

安裝與基本設定

安裝方法

npm 安裝
# 全域安裝 Claude Code
npm install -g @anthropic-ai/claude-code

# 驗證安裝
claude --version

# 檢查更新
npm update -g @anthropic-ai/claude-code
Docker 安裝
# 使用 Docker 運行
docker run -it --rm -v $(pwd):/workspace anthropic/claude-code

# 指定工作目錄
docker run -it --rm -v /path/to/project:/workspace anthropic/claude-code
從源碼安裝
# 克隆倉庫
git clone https://github.com/anthropics/claude-code.git
cd claude-code

# 安裝依賴
npm install

# 建置專案
npm run build

# 全域安裝
npm link

基本設定

環境變數設定
# 設定 API Key
export ANTHROPIC_API_KEY="your-api-key-here"

# 設定預設模型
export CLAUDE_MODEL="claude-3-sonnet-20240229"

# 設定輸出格式
export CLAUDE_OUTPUT_FORMAT="text"
專案初始化
# 初始化專案（建立 .claude 目錄）
claude /init

# 設定專案記憶體
claude /memory

# 查看專案設定
claude /config
權限設定
# 危險模式（跳過所有權限確認）
claude --dangerously-skip-permissions

# 設定特定權限
claude --permissions read,write,execute

# 查看當前權限
claude /permissions

使用技巧

  • 專案記憶體：使用 /memory 管理專案上下文
  • 檔案引用：使用 @filename 讓 Claude 讀取檔案內容
  • 目錄掃描：使用 @directory/ 讓 Claude 了解專案結構
  • 工具使用：Claude 可以執行檔案操作、搜尋、編輯等工具
  • 安全模式：預設會詢問權限，使用 --dangerously-skip-permissions 跳過

Shell 常用指令超級詳細指南

基本檔案系統操作

pwd (Print Working Directory)

功能: 顯示當前工作目錄的完整路徑

pwd                    # 顯示當前目錄
pwd -P                 # 顯示實際路徑(解析符號連結)
pwd -L                 # 顯示邏輯路徑(預設)

ls (List Directory Contents)

功能: 列出目錄內容，是最常用的指令之一

ls                     # 列出當前目錄內容
ls -l                  # 詳細格式(長格式)
ls -a                  # 顯示隱藏檔案
ls -lh                 # 人類可讀格式
ls -lt                 # 按修改時間排序
ls -R                  # 遞迴顯示

cd (Change Directory)

功能: 切換目錄

cd /path/to/directory  # 切換到指定目錄
cd ..                  # 回到上一層目錄
cd -                   # 回到上一個工作目錄

mkdir (Make Directory)

功能: 創建目錄

mkdir directory_name   # 創建目錄
mkdir -p path/to/deep/directory  # 遞迴創建多層目錄

檔案操作指令

touch

功能: 創建空檔案或更新檔案時間戳

touch filename         # 創建空檔案或更新時間戳

cp (Copy)

功能: 複製檔案和目錄

cp source destination   # 複製檔案
cp -r source_dir dest_dir  # 遞迴複製目錄

mv (Move/Rename)

功能: 移動或重新命名檔案和目錄

mv old_name new_name    # 重新命名
mv file directory/      # 移動檔案到目錄

rm (Remove)

功能: 刪除檔案和目錄

rm filename            # 刪除檔案
rm -r directory        # 遞迴刪除目錄
rm -rf directory       # 強制遞迴刪除(危險!)

檔案查看指令

cat (Concatenate)

功能: 顯示檔案內容

cat filename           # 顯示檔案內容
cat -n filename        # 顯示行號

less

功能: 分頁顯示檔案內容

less filename          # 分頁查看檔案
# 內部指令: q(退出), j/k(上下行), f/b(上下頁), /pattern(搜索)

head

功能: 顯示檔案開頭部分

head filename          # 顯示前10行
head -n 20 filename    # 顯示前20行

tail

功能: 顯示檔案結尾部分

tail filename          # 顯示最後10行
tail -f filename       # 持續監控檔案新增內容

搜索和過濾指令

grep (Global Regular Expression Print)

功能: 在檔案中搜索文本模式

grep "pattern" filename    # 在檔案中搜索
grep -i "pattern" filename # 忽略大小寫
grep -v "pattern" filename # 反向搜索
grep -r "pattern" directory # 遞迴搜索

find

功能: 在檔案系統中搜索檔案和目錄

find . -name "*.txt"      # 按名稱搜索
find /path -type f        # 只搜索檔案
find . -size +100M        # 搜索大於100MB的檔案
find . -mtime -7          # 搜索7天內修改的檔案

權限和所有權

chmod (Change Mode)

功能: 修改檔案和目錄權限

chmod 755 filename     # rwxr-xr-x
chmod u+x filename     # 給用戶添加執行權限
chmod -R 755 directory # 遞迴設定

chown (Change Owner)

功能: 修改檔案和目錄的所有者

chown user:group filename      # 改變所有者和群組
chown -R user:group directory  # 遞迴改變

壓縮和解壓縮

tar (Tape Archive)

功能: 打包和解包檔案

tar -czvf archive.tar.gz files # 創建gzip壓縮包
tar -xzvf archive.tar.gz       # 解壓gzip壓縮包
tar -cvf archive.tar files     # 僅打包不壓縮
tar -xvf archive.tar           # 解包

zip/unzip

功能: zip格式壓縮和解壓縮

zip -r archive.zip directory   # 遞迴壓縮目錄
unzip archive.zip              # 解壓zip檔案

系統監控和進程管理

ps (Process Status)

功能: 顯示運行中的進程

ps aux                 # 顯示所有進程
ps -ef                 # 顯示所有進程(另一種格式)
ps aux | grep pattern  # 搜索特定進程

top / htop

功能: 實時顯示系統進程

top                    # 啟動top
htop                   # 啟動htop(更友好)

kill

功能: 終止進程

kill PID               # 終止指定PID的進程
kill -9 PID            # 強制終止進程
killall process_name   # 終止所有同名進程

網路相關指令

ping

功能: 測試網路連通性

ping hostname          # 持續ping主機
ping -c 4 hostname     # ping 4次後停止

wget / curl

功能: 下載檔案 / 傳輸資料

wget URL               # 下載檔案
curl -O URL            # 下載檔案
curl -L URL            # 跟隨重定向

ssh (Secure Shell)

功能: 安全遠程連接

ssh username@hostname  # 連接到遠程主機
ssh -i keyfile user@host # 使用金鑰檔案

scp (Secure Copy)

功能: 安全檔案傳輸

scp file user@host:/path/    # 複製檔案到遠程
scp user@host:/path/file .   # 從遠程複製檔案

進階操作技巧

管道 (Pipe) |

功能: 將一個命令的輸出作為另一個命令的輸入

ls -l | grep ".txt"    # 列出所有.txt檔案
ps aux | grep firefox  # 查找firefox進程

重定向 (Redirection)

功能: 重定向輸入/輸出

command > file         # 重定向輸出到檔案(覆蓋)
command >> file        # 重定向輸出到檔案(追加)
command &> file        # 重定向所有輸出

命令組合

功能: 組合多個命令

command1 && command2   # command1成功後執行command2
command1 || command2   # command1失敗後執行command2
command1 ; command2    # 依序執行兩個命令
